## 설치 안내 (Installation Process)
1. 원격 저장소를 클론합니다.
```bash
$ git clone https://github.com/osamhack2020/WEB_InpenDansim_INPENDANSIM.git
```
2. npm install을 실행합니다.
```bash
$ npm install
```
3. npm start를 입력합니다.
```bash
$ npm start
```
4. http://localhost:8000 를 입력하여 사이트에 접속합니다.
